
DrPets is a comprehensive pet care platform built with Laravel (full-stack), designed to streamline pet ownership and service management. The application enables users to:

• Purchase pet products through an integrated e-commerce system

• Book veterinary appointments with real-time order tracking and status updates

• Register and manage your pets.

The project demonstrates my ability to design and implement scalable backend architectures, maintain clean and maintainable code, and handle complex business logic across multiple interconnected systems.

Key technical highlights include:
• Backend development: Laravel, REST APIs, database design, business logic

• Frontend integration: JS, Blade templates, dynamic interfaces, user-friendly experience

• Order tracking & appointments: Real-time state management and workflow processing

• Data modeling: Efficient handling of users, pets, orders, and appointments
